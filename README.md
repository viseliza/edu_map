# Дипломный проект  

#### Разрабатываемое мобильное приложение предназначено для структурированного представления учебной программы направления подготовки студентов колледжа. Программа обеспечивает наглядную визуализацию дисциплин и тем обучения по курсам и семестрам на основе официальных источников Министерства образования и внутренней учебной программы колледжа, а также предоставляет ссылки на дополнительные учебные ресурсы для углублённого самостоятельного изучения материала.

### Setup

Запустите докер контейнер сервера и запустите мобильное приложение через Android Panda:

```bash
# сборка зависимостей
docker-compose build --no-cache

# поднятие контейнера
docker-compose up
```

После запуска сервера необходимо зайти в документацию swagger и выполнить запрос на заполнение данных. Это заполнит данные групп, специальностей, учебных планов и предметов

[Хост сервера](http://localhost:18001) локальный, на порте 18001, endpoint /api/admin/fill
